Подробное описание

Qt клиент для Tinkoff invest API.

Данный класс предоставляет доступ к торговой платформе Тинькофф Инвестиции через grpc-интерфейс. В конструкторе класса необходимо передавать адрес сервера и токен. Все вызовы сервиса выполняются по адресу invest-public-api.tinkoff.ru:443 Токен можно получить в настройках своего профиля Тинькофф Инвестиций

Объявления и описания членов классов находятся в файлах:
  • investapiclient.h
  • investapiclient.cpp
